# ğŸ§© How to Run Locally (npm + Vite)

### Prerequisites
- Node.js (16+ recommended)  
- npm

### 1. Clone the repository
```bash
git clone https://github.com/smrasoul/Coffee-Online-Shop.git
cd Coffee-Online-Shop
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start development server
```bash
npm run dev
```
### 4. Build for production
```bash
npm run build
```
Creates a dist/ folder with optimized static files.

### 5. Preview the production build
```bash
npm run preview
```
Serves the dist folder so you can test the production output locally.

### Included npm scripts:
```bash
"scripts": {
  "dev": "vite",
  "build": "vite build",
  "preview": "vite preview"
}
```
